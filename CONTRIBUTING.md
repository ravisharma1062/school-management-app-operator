# Contributing

This repository is solo-maintained. Day-to-day development uses a direct-push workflow on the
default branch rather than pull requests — there is no dedicated review process for external
contributions today.

If you'd like to report a bug or suggest a change, please open an issue describing it. Pull
requests are welcome but may not be reviewed promptly, since this isn't currently set up for a
multi-contributor review cadence.
